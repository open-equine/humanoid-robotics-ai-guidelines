## About This Folder

Definition of Devices speaks about the classification of devices considered in the "Humanoids, Robotics and AI in Equine Stable Safety Guidelines". The guidelines documents in this repository reference to these devices. current version is 1.0.

## Device Categories Defined

| Category | Description |
|---|---|
| Humanoid Machines | Machines replicating human anatomical form with bipedal locomotion |
| Biomimetic Robotic Systems | Machines replicating the form or movement of any living biological organism |
| Aerial Systems | Autonomous or remotely operated machines capable of sustained flight |
| Ground Autonomous Machines | Autonomous wheeled, tracked, or legged machines that do not replicate biological form |
| Wearable and Attached Sensor Systems | Devices attached to horses, handlers, or stable infrastructure for data collection |
| Stationary Intelligent Systems | Fixed machines with AI processing capability and no autonomous locomotion |
| Algorithmic and Software Systems | Software-only systems that inform or influence stable management decisions |

## Quality-Control Folder

The Quality-Control folder contains the evidence that OE-GL-000 was tested before public release — defects logged, resolved, and verified. Institutions may execute the test cases as well. 

### Content-Quality

| File | Description |
|---|---|
| OE-GL-000-DOD-ContentQuality-v0_12.xlsx 
The file contains six tabs: Cover, Test Summary Report, Content Quality Test Execution, Content Quality Test Cases, Test Summary Report Template, and Change Log.

The blank Content Quality Test Cases tab is available for any party to independently verify the Definition of Devices document against the same 30 test cases.

For full details see the [Content-Quality README](Quality-Control/Content-Quality/README.md).
